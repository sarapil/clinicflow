# FAQ — الأسئلة الشائعة

**Q: How do I set up practitioner schedules?**
A: Go to Clinical Settings > Doctor Schedule. Create one schedule per day per doctor.

**Q: Can I import existing patient records?**
A: Use Frappe's Data Import tool with the CF Patient template from Tools > Data Import.

**Q: How are appointment reminders sent?**
A: Via NotifyPro (required for SMS/WhatsApp). Daily task marks no-shows and sends reminders.

**Q: Can multiple branches share the same ClinicFlow instance?**
A: With Enterprise plan — multi-branch support via branch filter on all DocTypes.

---

**Q: How do I report a bug?**
A: Open an issue at https://github.com/sarapil/clinicflow/issues with the `bug` label.

**Q: Where is the support channel?**
A: Email support@arkan.it.com or visit https://arkan.it.com/support
