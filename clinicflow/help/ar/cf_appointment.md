# CF Appointment — حجز الموعد

نموذج **CF Appointment** يُدير جدولة مواعيد المرضى مع التحقق الفوري من توفر الأوقات.

## ## patient

رابط إلى سجل **CF Patient**. اكتب اسم المريض أو رقمه للبحث.

## ## doctor

حدد **الطبيب المعالج** (رابط إلى المستخدم). تُحمّل الأوقات المتاحة بناءً على جدول الطبيب.

## ## appointment_date

حدد **تاريخ** الموعد. يتم تفعيل التواريخ التي بها فترات متاحة فقط.

## ## slot

اختر **فترة زمنية** متاحة من جدول الطبيب. تُتحقق الفترات من الحجوزات الموجودة لمنع الازدواجية.

## ## appointment_type

حدد النوع:

- **مريض جديد** (أول زيارة)
- **متابعة** (مريض عائد)
- **طارئ** (عاجل، يتجاوز الطابور العادي)
- **طب عن بُعد** (استشارة مرئية)

## ## status (الحالة)

| الحالة      | المعنى                 |
| ----------- | ---------------------- |
| Scheduled   | تم حجز الموعد          |
| Checked In  | وصل المريض إلى العيادة |
| In Progress | الطبيب يرى المريض      |
| Completed   | انتهت الاستشارة        |
| Cancelled   | تم إلغاء الموعد        |
| No Show     | لم يصل المريض          |

## سير العمل

1. المستقبل يحجز الموعد → الحالة: **Scheduled**
2. المريض يصل → انقر "تسجيل الوصول" → الحالة: **Checked In**
3. الطبيب يبدأ الاستشارة → الحالة: **In Progress**
4. حفظ الاستشارة → الحالة: **Completed**
