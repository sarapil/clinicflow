# ClinicFlow — كلينيك فلو

ClinicFlow هو **نظام متكامل لإدارة العيادات الخارجية** مصمم للمنشآت الطبية الحديثة. يغطي كل خطوة في رحلة المريض من التسجيل إلى الفوترة.

## الوحدات الرئيسية

| الوحدة                      | الوظيفة                                                         |
| --------------------------- | --------------------------------------------------------------- |
| **إدارة المرضى**            | تسجيل المرضى وإدارة التاريخ الطبي والحساسيات والأمراض المزمنة   |
| **المواعيد**                | حجز وإعادة جدولة وإدارة المواعيد والطوابير                      |
| **الاستشارة (السجل الطبي)** | تسجيل العلامات الحيوية والتشخيصات (رموز ICD) والوصفات والإحالات |
| **المختبر**                 | طلب الفحوصات المخبرية (رموز LOINC) وتسجيل ومشاركة النتائج       |
| **الأشعة**                  | طلب دراسات التصوير وتسجيل نتائج الأشعة                          |
| **الفوترة**                 | إنشاء الفواتير وتسجيل المدفوعات وإدارة مطالبات التأمين          |
| **الصيدلية**                | تتبع مخزون الأدوية وتسجيل الصرف مقابل الوصفات                   |

## البدء السريع

1. **تكوين الإعدادات** → اذهب إلى CF Clinical Settings وحدد مدة الاستشارة وإعدادات الرسائل القصيرة
2. **إضافة الأطباء** → قم بإعداد جداول الأطباء مع فترات الوقت المتاحة
3. **تسجيل المرضى** → استخدم نموذج CF Patient لتسجيل المرضى الجدد
4. **حجز المواعيد** → استخدم سير عمل حجز المواعيد

## مزودو التأمين المدعومون (نيجيريا)

- NHIA (الهيئة الوطنية للتأمين الصحي)
- Hygeia HMO
- Reliance HMO
- THT (Total Health Trust)
- AIICO Medical
