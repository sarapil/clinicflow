# CF Consultation — السجل الطبي الإلكتروني (EMR)

**CF Consultation** هو مستند السجل الطبي الأساسي. يلتقط اللقاء السريري الكامل: العلامات الحيوية والتشخيص والطلبات وخطة العلاج.

## ## patient

رابط إلى **CF Patient** الذي يُرى.

## ## vital_signs (العلامات الحيوية)

سجّل في النموذج الفرعي:

- ضغط الدم (انقباضي/انبساطي)
- درجة الحرارة (°م أو °ف)
- معدل النبض (نبضة/دقيقة)
- معدل التنفس
- تشبع الأكسجين (SpO₂ %)
- الوزن والطول (حساب BMI تلقائي)

## ## diagnosis_items (جدول التشخيصات)

أضف تشخيصاً واحداً أو أكثر مرتبطاً برموز ICD-10:

- ابحث بالرمز أو الوصف (يستخدم قاعدة بيانات MedCode ICD)
- حدد: أساسي / ثانوي / استبعاد / عمل

## سير العمل

1. يفتح الطبيب الاستشارة → يُدخل العلامات الحيوية
2. يسجّل الشكاوى الرئيسية ونتائج الفحص
3. يضيف التشخيصات برموز ICD-10
4. يُنشئ وصفة طبية (إذا لزم) → CF Prescription
5. يطلب فحوصات مخبرية (إذا لزم) → CF Lab Order
6. يحفظ ويُقدّم الاستشارة → إنشاء فاتورة تلقائياً
