# ClinicFlow

ClinicFlow is a **complete outpatient clinic management system** designed for modern medical facilities. It covers every step of the patient journey from registration to billing.

## Key Modules

| Module                 | What it does                                                             |
| ---------------------- | ------------------------------------------------------------------------ |
| **Patient Management** | Register patients, manage medical history, allergies, chronic conditions |
| **Appointments**       | Book, reschedule, and manage appointment slots and queues                |
| **Consultation (EMR)** | Record vital signs, diagnoses (ICD codes), prescriptions, referrals      |
| **Laboratory**         | Order lab tests (LOINC codes), record and share results                  |
| **Radiology**          | Order imaging studies, record radiologist findings                       |
| **Billing**            | Generate invoices, record payments, manage insurance claims              |
| **Pharmacy**           | Track drug inventory, record dispensing against prescriptions            |

## Getting Started

1. **Configure settings** → Go to CF Clinical Settings and set your consultation duration and SMS preferences
2. **Add doctors** → Set up Doctor Schedules with available time slots
3. **Register patients** → Use the CF Patient form to register new patients
4. **Book appointments** → Use the appointment booking workflow

## Supported Insurance Providers (Nigeria)

- Hygeia HMO
- Reliance HMO
- THT (Total Health Trust)
- AIICO Medical
