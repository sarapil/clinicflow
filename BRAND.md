# ClinicFlow — Brand Guidelines

> **كلينك فلو**

## 🎨 Color Palette

| Token | Value | Usage |
|-------|-------|-------|
| `--clinicflow-primary` | `#EF4444` | Primary brand color |
| `--clinicflow-primary-light` | `#EF444420` | Backgrounds, hover states |
| `--clinicflow-text` | `#1e293b` | Primary text |

## 🔤 Typography

- **Headings:** Inter (EN) / IBM Plex Sans Arabic (AR)
- **Body:** Inter (EN) / IBM Plex Sans Arabic (AR)
- **Code:** JetBrains Mono

## 📐 Logo Usage

| Variant | File | Usage |
|---------|------|-------|
| Full logo | `images/clinicflow-logo.svg` | App screen, about page |
| Desktop solid | `icons/desktop_icons/solid/` | Workspace (focused) |
| Desktop subtle | `icons/desktop_icons/subtle/` | Workspace (unfocused) |

## 🎯 Icon Style

- **Library:** Tabler Icons v3.30+ via `frappe.visual.icons`
- **Style:** Outlined, 1.5px stroke
