# ClinicFlow — Technical Context

## Overview

**Full-featured clinic management system.** ClinicFlow (كلينك فلو) is part of the Arkan Lab Frappe ecosystem.

- **Publisher:** Arkan Lab
- **Version:** 0.0.1
- **License:** GPL-3.0
- **Color:** `#EF4444` (Red)
- **Dependencies:** `frappe`, `frappe_visual`, `arkan_help`, `caps`, `medcode`

## Architecture

- **Framework:** Frappe v16
- **DocTypes:** 8
- **Prefix:** CF
- **Tier:** Healthcare

## Key Features

- Patient registration and medical records
- Appointment scheduling with visual calendar
- Prescription management with drug database
- Lab order management with results tracking
- Billing and insurance claim processing
- Clinical reporting and analytics
- MedCode integration for ICD/LOINC coding
