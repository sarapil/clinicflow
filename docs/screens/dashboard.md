# ClinicFlow — Dashboard Screen
# كلينك فلو — شاشة لوحة التحكم

## Overview
Main workspace dashboard for ClinicFlow.

## frappe_visual Components
- `frappe.visual.templates.dashboard()` — KPI cards
- `frappe.visual.sparkline()` — Trend indicators
- `frappe.visual.donut()` — Distribution charts

## CSS Effects
- `.fv-fx-glass` — Glassmorphism cards
- `.fv-fx-hover-lift` — Card hover animation
- `.fv-fx-page-enter` — Entrance animation

## Responsive Breakpoints
| Breakpoint | Layout |
|-----------|--------|
| Desktop (1024px+) | 3-column grid |
| Tablet (768px-1023px) | 2-column grid |
| Mobile (320px-767px) | Single column stack |
