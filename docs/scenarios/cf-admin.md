# CF Admin — Usage Scenarios
# سيناريوهات استخدام CF Admin

## Role Overview
- **Title**: CF Admin
- **App**: ClinicFlow
- **Primary DocTypes**: See docs/ARCHITECTURE.md
- **Device**: Desktop / Tablet / Mobile

## Daily Scenarios (يومي)

### DS-001: Dashboard Review
- **Goal**: Review key metrics and pending tasks
- **Pre-conditions**: User logged in with CF Admin permissions
- **Steps**:
  1. Navigate to ClinicFlow workspace
  2. Review dashboard KPIs
  3. Check notification badges
  4. Act on pending items
- **Breakpoints**: Desktop ✅ Tablet ✅ Mobile ✅

### DS-002: Create New Record
- **Goal**: Create a new record in the system
- **Steps**:
  1. Click "New" button from list view
  2. Fill required fields
  3. Save and submit if applicable
- **Breakpoints**: Desktop ✅ Tablet ✅ Mobile ⚠️

## Weekly Scenarios (أسبوعي)

### WS-001: Generate Reports
- **Goal**: Review weekly performance
- **Steps**:
  1. Navigate to Reports section
  2. Select date range
  3. Export to Excel/PDF

## Exception Scenarios (استثنائي)

### ES-001: Handle Errors
- **Goal**: Resolve system errors or data issues
- **Steps**:
  1. Check error message
  2. Verify permissions (CAPS capabilities)
  3. Clear cache if needed
  4. Contact admin if unresolved
